# Personal Website Mk1

A bare-bones personal website — no frameworks, no backend, just plain HTML, CSS, and a sprinkle of JavaScript. Built as a first pass to get something live and functional.

## What It Is

This is the Mk1 (first iteration) of my personal website. It's intentionally minimal — no React, no Node, no database. Just static files served up as-is. The goal wasn't to impress with tech, it was to have a clean, working site that I can point people to.

## Features

- Sidebar navigation
- About me page with shields.io tech badges
- Projects showcase with links to repos and live demos
- Contact page with GitHub, LinkedIn, and email
- Downloadable CV
- Visitor counter
- Fully responsive on mobile

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Deployed on Google App Engine
